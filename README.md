## Instrucciones de Uso

Dato: Este proyecto se complementa con Proyecto_Ejemplo_Back_End_1.

1. Clonar el proyecto.
2. Navegar a la carpeta del proyecto ```cd nombre-carpeta-proyecto```. 
3. Ejecutar ```ng serve -o``` para compilar y servir el proyecto en un servidor de desarrollo, y abrirlo automáticamente en el navegador predeterminado.
4. Rellenar el formulario y presionar el botón 'Agregar' para sumar el contenido a la tabla 'empleados' de la base de datos creada con los querys presentes en el archivo 'Query_DB.sql' presente en la carpeta raíz de 'Proyecto_Ejemplo_Back_End_1'.
